Caixa de Mercado em Java

Projeto em Java (terminal) com regras de caixa, estoque e vendas.

Java 21 Maven POO

Funcionalidades

  • Cadastro de produto com SKU, preco e estoque
  • Reposicao de estoque
  • Abertura de venda e adicao de itens
  • Finalizacao com desconto e forma de pagamento
  • Calculo de troco em pagamentos em dinheiro
  • Historico de vendas

Como executar

cd projects/caixa-mercado-java
mvn exec:java

Codigo fonte: src/main/java/com/matheus/caixa

Fluxo no terminal (exemplo)

===== CAIXA DE MERCADO =====
1. Listar estoque
2. Cadastrar novo produto
3. Repor estoque
4. Adicionar item a venda atual
5. Ver venda atual
6. Finalizar venda
...
